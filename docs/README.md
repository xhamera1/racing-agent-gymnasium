# `docs/` — reference material

Static snapshots of the upstream documentation we built against, plus the
official project brief. Keeping a local copy means our citations in
`reports/final_report.pdf` survive even if the upstream pages move.

- `CarRacingDocumentation.txt` — verbatim copy of
  <https://gymnasium.farama.org/environments/box2d/car_racing/>
- `SACDocumentation.txt` — verbatim copy of
  <https://stable-baselines3.readthedocs.io/en/master/modules/sac.html>
- `Projekt 5 Przestrzenie ciagle.pdf` — official task description (lecturer).

For implementation guidance, see `PLAN.md` and `IDEA.md` at the repository
root.
